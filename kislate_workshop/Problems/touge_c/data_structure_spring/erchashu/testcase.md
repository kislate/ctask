
## 结构直观图

假设输入如下先序序列（带空枝）：

```
1 A 2 B 4 D 0 null 0 null 5 E 0 null 0 null 3 C 0 null 6 F 0 null 0 null 0 null -1 null
```

其结构如下：

```
         1,A
        /   \
     2,B     3,C
    /   \      \
 4,D   5,E    6,F
```

- 1为根，2为左子，3为右子，2有左4右5，3有右6
